# 名称
mount——挂载一个文件系统。

# 概览
vmstat \[options] \[<u>delay</u> \[<u>count</u>]]
# 说明
vmstat 报告有关进程、内存、分页、块 I/O、陷阱、磁盘和 CPU 活动的信息。 
首次生成的报告会给出自上次重启以来的平均值。后续报告则会提供长度为延迟（delay）的采样周期内的信息。无论哪种情况，进程和内存报告都是即时生成的。
